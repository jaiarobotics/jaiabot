JAIA Software Configuration

Software Status

 
This will show the current software group, the software version, and if there is a new software package available.

Software Update

This will update the software packages on your system.

Software Group To Follow (release, beta, continuous)

This will update the software group you will get the jaiabot-embedded package from.
The release group is the most stable code.
The beta group is code that has some testing in the field.
The continuous group is code that has minimal testing in the field.
